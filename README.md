# Han flashcards

App designed to train reading of Han characters
by having the user input the corresponding _pinyin_.

Requires working knowledge of _pinyin_.
Some spoken Chinese vocabulary is vastly helpful.
The goal is to tackle the most commonly used phrases first to
be able to read text encountered in everyday life.
The vocabulary lists are therefore ordered from most to
least common.

This app is particularly suited for foreign-raised Chinese
who speak a bit but are mostly illiterate (such as myself).
